### 資料庫相關
1.資料庫在跑完一段insert into 後，要按commit
2.有時eclipse 沒有辦法run時，若是跑資料庫，有可能是沒有commit的問題。
3.在資料庫上，直接按新刪修改時，要按commit。
4.資料讓它透明，把要查詢的資料一起查出來，但需要的欄位都要有，然後放在一起做判斷。
